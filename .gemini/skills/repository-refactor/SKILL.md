# Repository Refactor Skill

This skill performs large-scale repository refactoring.

## Goals

- improve architecture
- reduce complexity
- remove duplication
- improve maintainability

## Workflow

1. Scan repository structure.
2. Identify large modules.
3. Detect duplication and coupling.
4. Propose refactoring plan.
5. Apply safe incremental changes.

## Rules

- avoid breaking public APIs
- maintain backward compatibility when possible
- prioritize readability over cleverness

## Output

Provide:

- refactoring plan
- modified modules
- summary of improvements
