# Repository Documentation Generator Skill

This skill generates high-quality documentation for the repository.

## Goals

- improve developer onboarding
- document architecture and modules
- clarify project structure

## Documentation Types

Generate or improve:

- README
- architecture overview
- module documentation
- developer setup instructions
- contribution guidelines

## Workflow

1. Scan repository structure.
2. Identify main modules and directories.
3. Understand application purpose.
4. Extract important configuration details.
5. Generate documentation.

## Rules

- prioritize clarity
- avoid excessive technical jargon
- keep documentation concise
- structure documentation logically

## Output

Produce documentation including:

- project overview
- folder structure explanation
- architecture summary
- development workflow
