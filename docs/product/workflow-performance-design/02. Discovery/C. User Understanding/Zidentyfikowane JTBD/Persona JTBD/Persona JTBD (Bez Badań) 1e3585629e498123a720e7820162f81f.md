## 2. 🧠 Profil Psychograficzny (Model DISC)

*Na podstawie powyższych danych, który typ osobowości dominuje w tym segmencie?*

### [ ] 🔴 Czerwony (Zdecydowany / "Do it Now")
**Sygnały**: Krótkie sesje, szybkie zakupy, wysoka konwersja z reklam "Promocja".
- **Główna obawa**: Strata czasu, skomplikowany proces.
- **Taktyka Contentowa**:
    - [ ] "Kup jednym kliknięciem"
    - [ ] "Wysyłka w 24h"
    - [ ] Gwarancja satysfakcji (usuń ryzyko decyzji)

### [ ] 🟡 Żółty (Społeczny / "Fun & Fame")
**Sygnały**: Dużo udostępnień, wejścia z Social Media, interakcje z wideo.
- **Główna obawa**: Bycie niemodnym, brak akceptacji grupy, nuda.
- **Taktyka Contentowa**:
    - [ ] Social Proof (zdjęcia ludzi, nie produktów)
    - [ ] "Bestseller", "Hit Instagrama"
    - [ ] Storytelling marki

### [ ] 🔵 Niebieski (Analityczny / "Details Matter")
**Sygnały**: Długi czas sesji, czytanie sekcji "Specyfikacja", pobieranie PDF.
- **Główna obawa**: Pomyłka, brak danych, słaba jakość.
- **Taktyka Contentowa**:
    - [ ] Tabele porównawcze
    - [ ] Szczegółowe dane techniczne
    - [ ] Wideo z recenzją/testem
    - [ ] Logiczne argumenty (ROI, oszczędność)

### [ ] 🟢 Zielony (Bezpieczny / "Stability First")
**Sygnały**: Powroty na stronę "O nas", czytanie regulaminów, szukanie kontaktu.
- **Główna obawa**: Zmiana, ryzyko, brak wsparcia, oszustwo.
- **Taktyka Contentowa**:
    - [ ] Certyfikaty i nagrody
    - [ ] "Jesteśmy na rynku od 10 lat"
    - [ ] Gwarancja zwrotu 365 dni
    - [ ] Case Studies (dowód sukcesu innych)
