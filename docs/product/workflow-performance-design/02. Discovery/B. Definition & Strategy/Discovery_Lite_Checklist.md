# Discovery Lite Checklist (4h Process)

> **Cel:** Szybka weryfikacja sensu biznesowego dla małych projektów / MVP.
> **Czas:** max 4 godziny.
> **Output:** Decyzja Go/No-Go + Zakres MVP.

## 1. Business Context (30 min)
- [ ] **Problem Statement:** Jaki *jeden* problem rozwiązujemy? (Językiem klienta).
- [ ] **Target Audience:** Kto za to zapłaci? (Konkretna nisza, nie "wszyscy").
- [ ] **Business Goal:** Po czym poznamy sukces za 3 miesiące? (1 metryka).

## 2. Competitive Landscape (30 min)
- [ ] **Top 3 Konkurentów:** Sprawdź ich pricing i opinie (1-2 gwiazdki na Trustpilot/AppStore).
- [ ] **Luka (The Gap):** Czego oni nie robią, a my możemy? (Szybkość? Obsługa? Nisza?).

## 3. Solution & Scope (60 min)
- [ ] **Core Value Proposition:** Jedno zdanie: "Pomagamy [X] zrobić [Y] dzięki [Z]".
- [ ] **Fat Marker Sketch:** Rozrysuj główny flow na kartce (max 3 ekrany).
- [ ] **Anti-Scope:** Czego NIE robimy w wersji v1? (Lista "Nice to have").

## 4. Technical Reality Check (30 min)
- [ ] **Data Sources:** Czy mamy dane? Czy API jest dostępne?
- [ ] **Tech Stack:** Czy standardowy stack (Next.js/Supabase) wystarczy? (Unikaj eksperymentów).
- [ ] **Legal/Risks:** RODO, płatności, regulacje.

## 5. Decision (30 min)
- [ ] **Go / No-Go?**
- [ ] **Estymacja:** Czy zrobimy to w budżecie klienta? (T-shirt sizing: S/M/L).

---
**Zasada:** Jeśli nie potrafisz wypełnić tego w 4h, projekt jest zbyt skomplikowany lub niejasny. Wróć do definicji problemu.
