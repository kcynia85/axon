# AI Readiness Assessment (Ocena Gotowości na AI)

> **Cel:** Sprawdzenie, czy organizacja/klient jest gotowy na wdrożenie AI. Pozwala uniknąć porażki przez brak danych lub kultury.
> **Baza wiedzy:** [5 Poziomów Dojrzałości](../../Zasoby/AI%20&%20Business%20Integration/Strategia%20i%20Modele/Strategia%20i%20Poziomy%20Dojrzalosci.md), [Model ADKAR](../../Zasoby/AI%20&%20Business%20Integration/Wdrożenie%20i%20Zarządzanie%20Zmianą/Zarzadzanie%20Zmiana%20ADKAR.md)

## 1. Gotowość Danych (Paliwo)
*Bez dobrych danych AI to tylko generator losowych treści.*

*   [ ] **Dostępność:** Czy dane są cyfrowe? (Nie w segregatorach, nie w głowach).
*   [ ] **Jakość:** Czy są kompletne i uporządkowane? (Brak pustych pól, jednolity format).
*   [ ] **Ilość:** Czy jest ich wystarczająco dużo do wyciągnięcia wniosków/nauczenia modelu?
*   [ ] **Prawność:** Czy mamy zgodę na ich przetwarzanie przez zewnętrzne AI?

## 2. Gotowość Technologiczna (Infrastruktura)
*   [ ] **Narzędzia:** Czy firma korzysta z chmury/SaaS, czy z zamkniętych systemów legacy?
*   [ ] **Integracje:** Czy systemy mają API?
*   [ ] **Bezpieczeństwo:** Czy polityka IT pozwala na użycie narzędzi typu OpenAI/Gemini?

## 3. Gotowość Ludzka i Kulturowa (ADKAR)
*   [ ] **Świadomość (A):** Czy zarząd/lider rozumie, że AI to nie magia, tylko narzędzie?
*   [ ] **Chęć (D):** Czy zespół widzi korzyść dla siebie (mniej nudy), czy czuje zagrożenie (zwolnienia)?
*   [ ] **Umiejętności (K/A):** Czy w zespole jest ktoś "techniczny" lub chętny do nauki promptingu?

## 4. Gotowość Biznesowa (Strategia)
*   [ ] **Problem:** Czy mamy zdefiniowany konkretny problem do rozwiązania? (Nie "wdrożmy AI", tylko "skróćmy czas X").
*   [ ] **Budżet:** Czy jest budżet na narzędzia, API i (ważne!) czas pracowników na naukę?
*   [ ] **Ryzyko:** Czy firma akceptuje margines błędu na początku?

---

## Wynik / Diagnoza

*   **Większość "NIE":** Firma nie jest gotowa. Zacznij od porządkowania danych i procesów (Poziom 0).
*   **Mieszane:** Zacznij od prostych narzędzi "pudełkowych" (ChatGPT) dla pojedynczych pracowników (Poziom 1).
*   **Większość "TAK":** Można planować automatyzację i integracje (Poziom 2/3).
