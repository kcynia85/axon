# E-commerce Hypotheses (Case Studies)

> **Zbiór sprawdzonych hipotez badawczych.**
> Użyj ich jako inspiracji do własnych testów A/B.

---

## 1. Bundle Feature (Zestawy)

### Hipoteza
| Element | Treść |
| :--- | :--- |
| **Jeśli** | Damy użytkownikom na PDP możliwość zakupu produktu wraz z innym dopasowanym produktem (Zestaw). |
| **To** | Średni przychód na użytkownika (ARPU) wzrośnie o 1%. |
| **Ponieważ** | Zadziała **Bundle Pricing Bias** – użytkownicy oceniają wartość zestawu wyżej niż sumę składników. |

### Wyniki (Case Study)
*   **Poziom ufności (Confidence):** 85.47% (Poniżej standardu 95%, ale akceptowalne przy niskim ryzyku).
*   **Moc (Power):** 80%.
*   **Wynik KPI:** +0.91% ARPU.

### Wnioski
Warto wdrożyć, jeśli koszt jest niski. Jeśli wysoki – powtórzyć test na większej próbie dla pewności 95%.

## 2. Pageviews Counter (Social Proof)

### Hipoteza
| Element | Treść |
| :--- | :--- |
| **Jeśli** | Pokażemy na PDP liczbę osób, które oglądały ten produkt ("120 osób ogląda"). |
| **To** | Konwersja (CR) wzrośnie o 3%. |
| **Ponieważ** | Zadziała **Social Proof** i **Unity Bias** (owczy pęd). Zmniejsza to niepewność zakupową. |

### Wyniki (Case Study)
*   **Poziom ufności (Confidence):** 87.69%.
*   **Moc (Power):** 80%.
*   **Wynik KPI:** +6.98% Konwersji (Znaczący wzrost!).

### Wnioski
Bardzo silny sygnał. Mimo Confidence < 95%, wysoki wzrost KPI (prawie 7%) sugeruje szybkie wdrożenie lub krótki test potwierdzający.

## 3. PayPal Default (Cognitive Load)

### Hipoteza
| Element | Treść |
| :--- | :--- |
| **Jeśli** | Domyślnie zaznaczymy PayPal w checkoutcie (jako najczęstszą metodę). |
| **To** | ARPU wzrośnie o 2%. |
| **Ponieważ** | Oszczędza to zasoby poznawcze (kliknięcia, myślenie). Redukcja tarcia. |

### Wyniki (Case Study)
*   **Poziom ufności (Confidence):** 90%.
*   **Moc (Power):** 80%.
*   **Wynik KPI:** +1.81% Konwersji.

### Wnioski
Wynik pozytywny, ale nie spektakularny. Warto wdrożyć (niski koszt zmiany - tylko checked="true"), ale monitorować, czy nie irytuje użytkowników płacących kartą.
