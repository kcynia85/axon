# Checklista

<aside>

**Pytania na Początek**

- [ ]  Jaki cel chcę osiągnąć?
- [ ]  O czym jest mój produkt/usługa?
- [ ]  Jakie problemy rozwiązuje?
- [ ]  Jak mogę to pokazać (udowodnić)?
- [ ]  Które informacje są bardzo ważne, a które mniej?
- [ ]  Jakie wrażenie chcę wywrzeć na samym początku?
- [ ]  Z jakim wrażeniem chcę pozostawić użytkownika (na koniec)?

**Pytania na Koniec (Review / Ocena)**

- [ ]  Czy mój LP (Landing Page) jest jasnym przedłużeniem narracji w reklamie?
- [ ]  Czy pierwszy ekran mówi wystarczająco dużo?
- [ ]  Czy produkt / usługa jest jasno przedstawiony?
- [ ]  Czy nawigacja po stronie jest prawdziwie intuicyjna?
- [ ]  Czy formularz działa (technicznie i pod kątem UX)?
- [ ]  Czy mam (uwzględniłem/am) miejsce na regulamin, RODO, zgody?
- [ ]  Czy mogę powiedzieć/napisać to jeszcze prościej?
- [ ]  Czy wystarczająco zachęcam do podjęcia działania (CTA)?
- [ ]  Czy użytkownik wie, co będzie dalej (po kliknięciu CTA, wypełnieniu formularza itp.)?
</aside>
