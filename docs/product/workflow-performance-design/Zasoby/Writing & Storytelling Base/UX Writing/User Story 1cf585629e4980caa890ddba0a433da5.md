### User story w skrócie

<aside>

1. Proofreading i copy editing.
2. Czekamy do zakończenia testów Usability, aby wprowadzić ewentualne aktualizacje:
    - zmiana części komunikacji
    - sparafrazowanie lub usunięcie komunikacji
3. Implementacja i bieżąca weryfikacja wdrożonego tekstu.
4. Lokalizacja i dostosowanie do innych rynków:
    - konsultacja i dodatkowe rekomendacje dla osób tłumaczących
    - może być konieczna parafraza tekstu źródłowego
    

</aside>
