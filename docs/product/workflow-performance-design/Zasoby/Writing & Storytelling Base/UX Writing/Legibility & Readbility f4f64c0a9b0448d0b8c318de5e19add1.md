# Legibility & Readbility

<aside>

| **Aspekt** | **Legibility (Czytelność)** | **Readability (Przystępność Tekstu)** |
| --- | --- | --- |
| **Definicja** | Jak łatwo można zidentyfikować poszczególne litery i słowa. | Jak łatwo tekst można zrozumieć i przyswoić. |
| **Skupia się na** | Fizycznych aspektach tekstu, takich jak czcionka i kontrast. | Organizacji treści, strukturze zdania i języku. |
| **Czynniki** | - Rozmiar czcionki<br>- Styl czcionki<br>- Kontrast<br>- Odstępy między literami/wierszami | - Struktura zdania<br>- Język<br>- Logiczna organizacja<br>- Użycie nagłówków i list |
| **Best Practices** | - Używać czcionek o dużej czytelności, np. bezszeryfowych w kontekście cyfrowym.<br>- Zapewniać wystarczający kontrast między tekstem a tłem.<br>- Dostosowywać wielkość czcionki i odstępy do kontekstu (np. większe dla czytania na urządzeniach mobilnych). | - Pisać w prostym, zrozumiałym języku, unikając żargonu.<br>- Używać krótkich akapitów i zdani.<br>- Logicznie organizować treść, używać nagłówków do strukturyzacji.<br>- Ułatwiać skanowanie treści poprzez użycie list, wypunktowań, nagłówków. |
| **Cel** | Ułatwienie identyfikacji znaków i słów. | Ułatwienie zrozumienia i przyswojenia informacji. |
|  |  |  |

</aside>
