# Analityka — 10 Use Cases

<aside>

1. Tworzenie pojedynczych źródeł prawdy. Wszystkie aplikacje w jednym widoku (Marketo, HubSpot, Salesforce, G4, [customer.io](http://customer.io)). Kluczowe kompetencje: Czyszczenie Danych, Analiza Danych, Programowanie. Kluczowe narzędzia: Python, R, SQL
2. Historie danych z zaawansowaną wizualizacją. Uczenie maszynowe do prognozowania. Kluczowe kompetencje: Czyszczenie Danych, Analiza Danych. Kluczowe narzędzia: Python, R, Power BI, Tableau.
3. Demokratyzacja danych poprzez interaktywne dashboardy.
4. Tworzenie lepszych kampanii dzięki segmentacji.
5. Przewidywanie odpływu klientów.
6. Przewidywanie wartości życiowej klienta.
7. Wykorzystanie eksperymentów na dużą skalę.
8. Analiza sentymentu w mediach społecznościowych.
9. Ustanawianie lepszych celów i dokładniejsze prognozowanie.
10. Automatyzacja analizy danych z ankiet.
