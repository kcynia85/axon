# Use Case: Sklep internetowy z elektroniką

<aside>

**Schemat organizacyjny** - klasyfikacja produktów według kategorii:

- Elektronika
- Odzież
- Artykuły spożywcze
- Książki
- Dom i ogród

**Struktura organizacyjna** - hierarchiczne powiązania między kategoriami:

- Elektronika
    - Telefony
        - Smartfony
        - Telefony stacjonarne
    - Laptopy
        - Laptopy gamingowe
        - Laptopy biznesowe
    - AGD
        - Małe AGD
        - Duże AGD

W tym przykładzie schemat organizacyjny grupuje produkty według ich podstawowych cech (elektronika, odzież, itp.), co pomaga użytkownikom zorientować się w głównych kategoriach dostępnych w sklepie. Natomiast struktura organizacyjna tworzy hierarchiczne powiązania między kategoriami i podkategoriami, umożliwiając precyzyjną nawigację i wyszukiwanie produktów.

Ten system organizacyjny zapewnia użytkownikom przejrzystą "mapę" do nawigacji w sklepie, ułatwiając znalezienie potrzebnych produktów.

</aside>
