# Modyfikatory

### **Kwalifikatory (Qualifiers)**

Określają zakres, część, uogólnienie.

```
❌ Popraw UX strony.
✅ Zaproponuj kilka drobnych usprawnień UX strony głównej sklepu.

❌ Jakie błędy UX popełniono?
✅ Jakie najczęstsze błędy UX zauważasz w formularzu rejestracji?

❌ Zoptymalizuj treść.
✅ Zoptymalizuj większość sekcji opisowych bez zmiany nagłówków.

```

### **Przymiotniki (Adjectives)**

Określają typ, ton lub styl oczekiwanej treści.

```
❌ Stwórz tekst do aplikacji.
✅ Stwórz **kreatywny** tekst onboardingowy do aplikacji do nauki języków.

❌ Daj nazwę dla firmy.
✅ Zaproponuj **profesjonalną** nazwę dla agencji konsultingowej.

❌ Zrób opis.
✅ Napisz **zwięzły** opis nowej funkcji w panelu użytkownika.

```

### **Przysłówki (Adverbs)**

Mówią **jak** coś zrobić.

```
❌ Wyjaśnij, o co chodzi.
✅ Wyjaśnij **dokładnie** działanie nowej funkcji.

❌ Przepisz to.
✅ Przepisz to **zwięźle i jasno**, aby było zrozumiałe dla laików.

❌ Napisz opis.
✅ Napisz opis **sprytnie**, z lekkim humorem i grą słów.

```

### **Intensyfikatory (Intensifiers)**

Podbijają wagę, oczekiwania.

```
❌ Napisz hasło reklamowe.
✅ Potrzebuję **niezwykle** chwytliwego hasła reklamowego dla kampanii outdoorowej.

❌ Daj pomysł na CTA.
✅ Zaproponuj **absolutnie najlepszy** CTA button na stronie koszyka.

❌ Zrób tagline.
✅ Stwórz tagline, który będzie **naprawdę oryginalny** i wyróżni markę.

```

### **Negacje (Negatives)**

Określają **czego unikać**.

```
❌ Zrób opis produktu.
✅ Zrób opis produktu, **nie używając technicznego żargonu**.

❌ Popraw ten tekst.
✅ Popraw tekst, **unikając ogólników** i pustych fraz.

❌ Zrób alternatywę.
✅ Zrób alternatywną wersję, **bez powtarzania wcześniejszych pomysłów**.

```

### **Liczby (Number words)**

Precyzują **ile rzeczy chcesz**.

```
❌ Wymyśl CTA.
✅ Daj mi **3 propozycje** CTA do strony głównej.

❌ Napisz kilka nazw.
✅ Zaproponuj **5 nazw** dla nowej aplikacji do budżetowania.

❌ Podaj pomysły.
✅ Podaj **2 alternatywne wersje** onboardingowego ekranu.

```

### **Czas (Time words)**

Określają **terminy i czas wykonania**.

```
❌ Napisz wiadomość dla klienta.
✅ Napisz wiadomość dla klienta, którą mogę wysłać **dziś wieczorem**.

❌ Przygotuj brief.
✅ Przygotuj brief do testów A/B, który będzie gotowy **do jutra**.

❌ Wymyśl kampanię.
✅ Wymyśl kampanię na **nadchodzący tydzień** promocyjny.

```

### **Miejsce (Place words)**

Dodają **lokalny lub kulturowy kontekst**.

```
❌ Zrób opis funkcji.
✅ Zrób opis funkcji **z myślą o użytkownikach z Polski**.

❌ Stwórz kampanię reklamową.
✅ Stwórz kampanię reklamową **dopasowaną do rynku niemieckiego**.

❌ Zrób wersję językową.
✅ Zrób wersję tekstów **dla rynku hiszpańskojęzycznego**, z uwzględnieniem kultury.

```

### **Stopień (Degree words)**

Wskazują **jak duża ma być zmiana lub dostosowanie**.

```
❌ Zoptymalizuj UX.
✅ Wprowadź **drobne poprawki**, które zwiększą intuicyjność nawigacji.

❌ Zmień to.
✅ Zmiany są **zdecydowanie zbyt duże** – zaproponuj bardziej subtelną wersję.

❌ Popraw tekst.
✅ Tekst jest **lekko nieczytelny** – popraw go, zachowując styl.

```
