# E-commerce Automation Map

> **Powiązane Zasoby (Wiedza):**
> *   [Automatyzacja i Analityka](../../Zasoby/Ecommerce%20Knowledge%20Base/Technologia%20i%20Operacje/Automatyzacja%20i%20Analityka.md)

Mapa procesów w sklepie, które można (i trzeba) zautomatyzować.

## 1. Obsługa Klienta (CS)
*   [ ] **Statusy zamówień:** Automatyczne maile/SMS po zmianie statusu (Spakowane, Wysłane).
*   [ ] **Zwroty:** Formularz online generujący etykietę zwrotną (np. Wygodne Zwroty).
*   [ ] **FAQ:** Chatbot AI odpowiadający na pytania "Gdzie moja paczka?".

## 2. Marketing (Marketing Automation)
*   [ ] **Porzucony koszyk:** Sekwencja 3 maili (1h, 24h, 48h po porzuceniu).
*   [ ] **Welcome Pack:** Mail z rabatem po zapisie do newslettera.
*   [ ] **Win-back:** Mail do klientów, którzy nie kupili od 90 dni.
*   [ ] **Urodziny:** Automatyczny kod rabatowy w dniu urodzin.

## 3. Zarządzanie Produktem i Logistyka
*   [ ] **Stany magazynowe:** Synchronizacja w czasie rzeczywistym (Sklep <-> Allegro <-> Magazyn).
*   [ ] **Faktury:** Automatyczne wystawianie i wysyłka po opłaceniu zamówienia.
*   [ ] **Low Stock Alert:** Powiadomienie do działu zakupów, gdy towar się kończy.
