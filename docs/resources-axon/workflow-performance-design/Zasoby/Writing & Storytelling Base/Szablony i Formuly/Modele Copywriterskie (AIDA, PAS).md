# Model AIDA

### Zero-shot plus + Model AIDA

<aside>

```markdown
**Prompt:** Wygeneruj mi tekst marketingowy według modelu AIDA.
**Usługa/Produkt:** [np. Trener personalny dla młodych mam]
**Insight (Problem/Pragnienie klienta):** [np. Moje klientki to młode matki, które chcą być silniejsze i sprawniejsze. Chcą czuć się swobodnie podczas ćwiczeń, mając świadomość, że nikt nie patrzy na ich proces dochodzenia do formy.]
**CTA (Wezwanie do działania):** [np. Umów się na bezpłatną konsultację.]
```

### **D - Desire**

Celem każdej reklamy jest wyrwanie odbiorcy ze strefy komfortu - pokazanie mu, że ma problem / pragnienie, które są niezaspokojone.

### **A - Action**

Na końcu musisz jasno i precyzyjnie powiedzieć odbiorcy, co ma zrobić.

<aside>

- Elementy Strefy Domknięcia
    1. Podsumowanie oferty: Co dokładnie klient dostaje?
    2. Przypomnienie głównej korzyści Value Proposition
    3. Zbudowanie wartości Podbicie wartości, aby cena wydawała się niska.
    4. Bonusy Dodatkowe elementy, które zwiększają atrakcyjność oferty.
    5. Dowód społeczny Testimonials / Social Proof
    6. Poczucie pilności (FOMO) Ograniczenie czasowe lub ilościowe.
    7. Gwarancja / Uniknięcie ryzyka: "Nic nie ryzykujesz, masz 30 dni na zwrot".
