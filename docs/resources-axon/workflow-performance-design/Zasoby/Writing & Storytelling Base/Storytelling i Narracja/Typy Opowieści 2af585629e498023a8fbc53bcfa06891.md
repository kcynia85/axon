# Typy Opowieści

<aside>

- **Opowieści o firmie:**
    - **Wewnętrzne:** Kierowane do pracowników, budujące kulturę.
    - **Zewnętrzne:** W punktach styczności z marką.
- **Opowieści dla społeczności:** Dzielenie świata na "my" i "oni" buduje wspólnotę.
- **Opowieści o kimś ważnym:** O założycielu, kliencie, lokalnym bohaterze. Dają poczucie wspólnoty.
- **Opowieści o miejscu:** Podkreślają unikatowość lokalizacji.
- **Opowieści o wydarzeniu:** Budują narrację wokół kluczowych momentów.
- **Opowieści o nagrodach i karach:** Kształtują pojedyncze zachowania.
- **Opowieści o rytuałach:** Wyjaśniają "dlaczego robimy to w ten sposób".
</aside>
