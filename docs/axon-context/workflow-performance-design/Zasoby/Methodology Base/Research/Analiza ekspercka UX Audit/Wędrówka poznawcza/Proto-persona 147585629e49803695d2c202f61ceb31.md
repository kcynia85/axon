# Proto-persona

**Stanowisko:** Specjalistka ds. marketingu cyfrowego

**Firma:** Mała agencja marketingowa

**Obowiązki:** Analiza kampanii marketingowych, optymalizacja stron internetowych, raportowanie wyników klientom

</aside>

<aside>

**Zainteresowania:** Trendy w marketingu internetowym, analityka danych, narzędzia do optymalizacji SEO i UX

</aside>

<aside>

**Motywacje:**

1. Zwiększenie skuteczności kampanii
2. Poprawa widoczności stron klientów

4. Ułatwienie raportowania wyników dla klientów
</aside>

<aside>

**Frustracje:**

1. Złożoność narzędzi analitycznych

3. Zbyt długie czasy ładowania raportów
4. Skąpe możliwości dostosowania raportów do potrzeb klientów
</aside>

<aside>

**Cele:**

1. Uzyskiwanie przejrzystych danych o zachowaniach użytkowników

4. Lepsze współprace z klientami dzięki czytelnym raportom
