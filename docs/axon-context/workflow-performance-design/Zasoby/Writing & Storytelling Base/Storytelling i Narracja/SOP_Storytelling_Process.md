## **Algorytm Tworzenia (Proces)**

<aside>

1. **Wybierz temat (Rdzeń):**
    - **Prosty:** Dopasowanie metafor do odbiorcy.
    - **Wiarygodny:** Stworzenie stereotypowych bohaterów, Waluta społeczna (np. historia o ważnym, ciekawym miejscu, o miłości i powrocie), Cel, Napęd (zewnętrzna lub wewnętrzna motywacja), Imię.
2. **Stworzenie bohatera (Stereotyp):**
    - Cel: Niska wartość bohatera i wysoka wartość celu.
3. **Przefiltrowanie rzeczywistości:**
    - Wyrzucenie nudy i dodanie dramatu.
4. **Uwiarygodnienie rdzenia (Etos, Patos, Logos):**
    - Autorytet
    - Liczby
    - Badania
    - Rzeczywiste sytuacje
5. **Zaskoczenie:**
    - Ujawnienie prawdziwych motywów.
6. **Dodanie emocji.**
7. **Wybranie medium i formatu.**
</aside>

## **Niezbędne Elementy Historii (Składniki)**

<aside>

- [ ]  **Osadzenie w miejscu i czasie:** Kontekst jest królem.
- [ ]  **Główny bohater:** Osoba, firma, przedmiot lub autor. Użyj zwrotu *"Wyobraź sobie, że..."*, by uczynić czytelnika bohaterem.
- [ ]  **Imię bohatera:** Czyni go bardziej namacalnym.
- [ ]  **Przeszkoda / Wezwanie:** Problem lub "czarny charakter", z którym trzeba walczyć.
- [ ]  **Droga:** Wydarzenia, które prowadzą do zmiany.
- [ ]  **Autentyczność:** Przyznanie się do błędu buduje więź i wiarygodność.
- [ ]  **Odsłonięcie emocji:** Pokaż, co czuł bohater (nie tylko co robił).
- [ ]  **Dialogi:** Kilka linijek ożywia tekst i nadaje dynamikę.
- [ ]  **Mentorzy:** Ktoś, kto pomógł bohaterowi (często rola Twojej marki).
- [ ]  **Puenta:** Wniosek lub lekcja na koniec.
</aside>
