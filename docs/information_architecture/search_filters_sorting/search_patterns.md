## Local Search (MVP - Present in all Lists)
┌──────────────────────────────────────┐
│ [🔍 Search...]           [Filters ▾] │
└──────────────────────────────────────┘

Present in:
- Projects List → Search by name
- Spaces List → Search by name, description
- Agents List → Search by name, role, keywords
- Crews List → Search by name, goal
- Knowledge Hubs → Search by name
- Prompts → Search by name, role
- Automations → Search by name, description
- Services → Search by name
- Tools → Search by function name, display name
- LLM Models → Search by name, alias
- Inbox → Search by item name

Behavior:
- Real-time filtering (debounced 300ms)
- Highlights matching text in results
- Case-insensitive
- Searches multiple fields (name, description, keywords)
