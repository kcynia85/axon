## 1. The LIFT Model Framework
*Use this framework to audit and improve the page.*

### 🚀 Value Proposition (The "Why")
*Is it instantly clear what this is and why it's great?*
- [ ] **Headline:** Benefit-driven (e.g., "Sleep better" vs "Foam Mattress").
- [ ] **Visuals:** High-res photos + Context shots (Show usage, not just product).
- [ ] **Price Anchoring:** Show RRP if discounted to visualize savings.

### 🛡️ Anxiety Reduction (The "Trust")
*What is the user afraid of?*
- [ ] **Returns:** "Free 30-day returns" near the Buy Button.
- [ ] **Shipping:** "Free delivery by Tuesday" (Be specific).
- [ ] **Social Proof:** "4.8/5 based on 230 reviews" (Not just stars, show count).

### 🚦 Clarity & Relevance
*Is the path forward obvious?*
- [ ] **ATC Button:** Contrasting color, sticky on mobile.
    *   **Timing Psychologiczny:** Rozważ prezentowanie dodatkowych ofert (upsell/cross-sell) zaraz po kliknięciu "Dodaj do koszyka" – to [Moment Sukcesu](05.%20Growth%20&%20Market/C.%20Conversion%20&%20Retention/Gamification%20Framework.md#c-variable-reward) dla użytkownika i czas wyrzutu dopaminy.
- [ ] **Hierarchy:** Price -> Variants -> Button -> Assurance.
- [ ] **Fold:** Key info (Img, Title, Price, Button) visible without scrolling on mobile.

### 🌪️ Distraction & Urgency
*Are we focused?*
- [ ] **Remove:** Sidebar banners, unrelated cross-sells above the fold.
- [ ] **Add:** "Order in 2h for shipping today" (If true).
- [ ] **Subtelny Upselling:** Integruj oferty premium/dodatki dyskretnie, np. poprzez [Dyskretne Oznaczenia Premium](05.%20Growth%20&%20Market/C.%20Conversion%20&%20Retention/Gamification%20Framework.md#d-strategia-subtelnego-upsellingu-wewnątrz-aplikacji), unikając agresywnych pop-upów (patrz: [Strategia Subtelnego Upsellingu](05.%20Growth%20&%20Market/C.%20Conversion%20&%20Retention/Gamification%20Framework.md#strategia-subtelnego-upsellingu-wewnątrz-aplikacji)).

## 2. DISC Content Strategy
*Optimization for different psychotypes hitting the same page.*

| Psychotype | Focus Area | Optimization Tactic |
| :--- | :--- | :--- |
| **🔴 Red (Decisive)** | **Buy Box** | "Buy Now" button (skip cart), quick delivery date visible immediately. |
| **🟡 Yellow (Social)** | **Reviews/UGC** | Instagram photos section, video reviews, "Bestseller" badges. |
| **🔵 Blue (Analytical)** | **Tabs/Specs** | Detailed technical specs table, comparison charts, "How it works" diagrams. |
| **🟢 Green (Safe)** | **Footer/Trust** | Clearly visible warranty info, "Family business" story, secure payment icons. |

## 3. 🔪 Conversion Killers (Audit Checklist)
*If checked, fix immediately.*

- [ ] **Slow Load:** Images not optimized (Wait time > 3s).
- [ ] **Hidden Costs:** Shipping calculated only at checkout.
- [ ] **Stock Fear:** No info if item is in stock (or "Out of stock" hidden).
- [ ] **Mobile Touch:** Buttons too small (< 44px height).
- [ ] **Review Ghost Town:** 0 reviews visible (better to hide section than show "0 stars").
