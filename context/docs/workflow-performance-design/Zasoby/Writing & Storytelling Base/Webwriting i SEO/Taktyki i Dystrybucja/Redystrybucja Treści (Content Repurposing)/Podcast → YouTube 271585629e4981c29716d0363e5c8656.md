# Podcast → YouTube

<aside>

Spora część odbiorców słucha podcastów na YouTube, więc nawet jeśli nie nagrywasz wideocastów (podcast + obraz), to i tak warto je tam umieścić.

Tutaj również bardzo pomocne będzie [narzędzie](https://growthtools.pl/#/item/tubebuddy), o którym wspominałem przy krótkich formatach. Za jego pomocą stworzysz proste grafiki czy animacje, które pozwolą opublikować podcast na YT.

</aside>
